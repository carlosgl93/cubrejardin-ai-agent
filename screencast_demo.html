🌱 WhatsApp Business Management

Server-to-Server Authentication Flow - Technical Documentation for App Review

⚠️ Important: This is a server-to-server application. There is no end-user facing Meta login flow. Authentication is performed once by the business owner via Meta Business Manager.

Authentication Flow

1

Business Owner Login

Business owner logs into Meta Business Manager at business.facebook.com

2

System User Creation

Navigate to Business Settings → Users → System Users and create/select a system user

3

Assign WhatsApp Account

Grant the system user full control of the WhatsApp Business Account

4

Generate Access Token

Generate permanent token with whatsapp_business_management permission

whatsapp_business_management
5

Server Configuration

Token is securely stored in server environment variables (never exposed to clients)

Server Configuration

# Environment Variables (.env file) # Generated in Step 4 above FACEBOOK_PAGE_ACCESS_TOKEN="EAAGxxxxxxxxxxxxxxxx" # WhatsApp Phone Number ID from Meta Business Manager WHATSAPP_PHONE_NUMBER_ID="123456789012345" # App Secret for webhook signature validation FACEBOOK_APP_SECRET="abc123def456..." # Webhook verification token WHATSAPP_WEBHOOK_VERIFY_TOKEN="your_custom_token"

Permission Usage Demonstration

The whatsapp_business_management permission enables the following capabilities:

📥 Receive Messages

Webhook receives incoming customer messages from WhatsApp Cloud API

📤 Send Messages

Bot sends automated responses using the API with the access token

✅ Mark as Read

Messages are marked as read to provide better user experience

📋 Send Templates

Approved message templates sent outside 24-hour messaging window

🤝 Handoff to Agents

Transfer conversations to human agents via pass_thread_control

API Request Example

// Sending a message using whatsapp_business_management permission POST https://graph.facebook.com/v21.0/{phone_number_id}/messages Headers: Authorization: Bearer {FACEBOOK_PAGE_ACCESS_TOKEN} Content-Type: application/json Body: { "messaging_product": "whatsapp", "to": "+1234567890", "type": "text", "text": { "body": "El Tiqui Tiqui tiene un precio de $690 por planta..." } } // This request uses the whatsapp_business_management permission // granted to the access token in Meta Business Manager
